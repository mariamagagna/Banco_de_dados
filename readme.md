# Banco de dados 🎲

### 🧐 O que são banco de dados? 

São **dados coletados e armazenados** de forma **organizada** em um espaço eletrônico, **SGBD** (Sistema Gerenciador de Banco de Ddos) ou **DBMS** (Data Base Management)

---

### ❓Por que ele é importante? 

>**Persistência** - Capacidade dos dados de se manterem/sobreviverem a uma interrupção.

>**Manipulação** - Usa DML (Data Manipulation Language), subgrupo do SQL(Structured Query Language), para adicionar, atualizar, trocar dados.

>**Organização** - Maneira com que um Banco de Dados é "formatado"(relacional x não relacional).

O banco de dados é crucial para **armazenar, organizar e gerenciar grandes volumes de informações de forma segura e estruturada**, permitindo acesso rápido, consistência dos dados e suporte estratégico para a tomada de decisões.

---

### ❗O que é SGBD (Sistema Gerenciador de Banco de Dados)?

É como se fosse o **"gerente"**, ou seja, o **software responsável por organizar, manipular e proteger informações sensíveis** perdidas dentro do banco de dados. Sem ele, os programadores teriam de interagir diretamente com os bits e bytes de forma manual.

---

### 💁🏻‍♂️ Dados x Informação

#### Dados:

**Dados brutos** ou **_raw data_**  são **informações que são coletadas diretamente da fonte** e que não passaram por nenhum tipo de manipulação, limpeza ou processamento. Matéria-prima.

**_Principais características:_**
1. **Não processados:** Resultados primários.
2. **Desorganizados:** Não possuem ordem.
3. **Formato variado:** Podem vir em formatos de texto, número...
4. **Matéria-prima:** Funcionam como matéria-prima para a análise de dados.

Exemplo: 38, Maria... 
<br>

#### Informação:

A informação é o **resultado do processamento, organização e interpretação dos dados.** Pegar um número ou fato e **lhe dar contexto/propósito.** 

Exemplo: 38°C, 38 anos...
<br>

#### Linha de produção:

```
Dado bruto -> Processamento (contexto) -> 
Informação (útil) -> Conhecimento (aplicação)
```
