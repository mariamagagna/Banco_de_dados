Banco de Dados

Características/pilares do banco de dados:

  • Persistência e consistência: Entender como os dados permanecem salvos e corretos ao longo do tempo.
  • Integridade e segurança: Conhecer os mecanismos protegem e validam os dados.
  • Compartilhamento eficiente: Saber como múltiplos usuários trabalham simultaneamente.

  1. Persistência: Dado permanece salvo após desligar o sistema.
  2. Consistência: Dados seguem regras lógicas. Sistema impede informações impossíveis.
  3. Integridade: Garante relacionamentos entre diferentes dados (válidos/sentido).
  4. Segurança: Controle de vizualização e alteração de dados.
  5. Compartilhamento: Vizualizar, modificar e acessar dados (vários usuários).
