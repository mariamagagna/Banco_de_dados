Oque é arquitetura de Banco de Dados?

É como/forma que os dados são organizados/armazenados.

Pilares dos bancos de dados:

  1. Salvar
  2. Buscar
  3. Relacionar

Relacional

  • O que é? Dados organizados em tabelas que se relacionam entre si.
  • Cada tabela representa uma entidade específica.

    • Aluno 1 Aluno 2
    Nota 10 5
    Idade 16 17

    Linhas (tuplas): Registro único.

    Colunas (atributos): Tipo de informação.

  • Chaves primárias (ID):Identificador único que distingue de outros dados.
  • Evita duplicação de dados.
  • Exemplos:

    • Tabela Livro ...
    Nome ...
    Autor ...
    Empréstimo ...
    ID ...

    Tabela Produto ...
    ID ...
    Nome ...
    Preço ...
    Estoque ...

    Tabela Funcionários ...
    ID ...
    Nome ...
    Cargo ...
    Data Admissão ...
